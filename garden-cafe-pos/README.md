## 🎯 Overview

**Garden & Cafe POS** is a full-stack MERN application that revolutionizes restaurant operations with real-time order management, multi-role authentication, and automated billing with NPR tax calculations (10% Service Charge + 13% VAT).

### Why Garden & Cafe POS?

- ⚡ **Real-time Updates**: Socket.io powered live synchronization
- 🎨 **Luxury UI**: Dark theme with gold (#d4af37) accents
- 💰 **Multi-Payment**: Cash, eSewa, Khalti, QR Code, Tap to Pay
- 📊 **Analytics**: Comprehensive revenue and sales reports
- 🧾 **PDF Receipts**: Professional, auto-generated receipts
- 📱 **Responsive**: Works seamlessly on tablets, phones, and desktops
- 🔒 **Secure**: JWT authentication with role-based access control

## ✨ Features

### 👨‍💼 Admin Panel
- 📊 Real-time revenue dashboard
- 🍽️ Complete menu management (CRUD operations)
- 📈 Analytics & reports (daily, weekly, monthly)
- 💳 Payment method breakdown
- 🏆 Top-selling items tracking
- 👥 Staff management
- 🏪 Table configuration

### 🧑‍💼 Waiter Panel
- 🎯 Live table grid with real-time status
- 🛒 Smart order cart with session management
- 📝 Add/modify orders for occupied tables
- 💰 Bill settlement with multiple payment options
- 🧾 QR code generation for digital payments
- ⏱️ Table occupancy time tracking
- 👥 Collaborative table management

### 👨‍🍳 Kitchen Display System
- 🎫 Ticket-based order display
- 🔄 Status management (Pending → In Progress → Ready)
- 🔔 Real-time order notifications
- 📋 Special request notes
- 🎯 Filter by order status
- 📊 Active order statistics

### 💰 Financial Features
- ✅ Automatic 10% Service Charge calculation
- ✅ Automatic 13% VAT calculation
- ✅ Combo/Bundle pricing support
- ✅ Detailed billing breakdown
- ✅ Multiple payment methods
- ✅ Professional PDF receipt generation
